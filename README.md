# docker-hsxkpasswd

`hsxkpasswd` is a password generator. It is hereby shoved inside a Docker container.

## Installation

To build it locally.

```bash
$ docker build .
```

## Usage

Or simply pull it from DockerHub.

```bash
$ docker pull vanaxe/hsxkpasswd
$ docker run -it --rm hsxkpasswd
```
